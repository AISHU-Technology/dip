# 侧边栏布局

- 侧边栏从上到下分为 5 个部分

1. logo + shou 按钮
2. 应用菜单（my-app） + 钉住的应用
3. 应用商店（app-store）
4. AI Data Platform + 系统工作台
5. 用户头像 + 名称

## 布局细节

- 1 居顶部
- 2 和 3 按顺序往下排列，2 和 3 中间有分割线
- 4 和 5 居底部，4 和 5 中间有分割线
- 3 和 4 中间是空的

# 交互

- 展开宽度 `240px`, 点击收起后，宽度 `60px`，菜单项高度 `40px`
- 选中的菜单的图标是渐变色 `background: linear-gradient(210.47deg, #1C4DFA 17.63%, #3FA9F5 92.96%);`，文字是主题色，菜单项最前面宽度为 `2px` 的竖线，颜色是渐变色 `background: linear-gradient(180deg, #3FA9F5 0%, #126EE3 100%);`
- 选中的菜单项是钉住的应用，图标不变色，使用应用本身图标
- 菜单悬浮是默认悬浮背景色，选中后背景色是主题色的 0.2 透明度
- AI Data Platform + 系统工作台没有选中态，只有悬浮态，点击打开新标签页
- 点击 AI Data Platform 菜单 ，链接是`https://dip.aishu.cn/`
- 点击 系统工作台 菜单，链接是`https://anyshare.aishu.cn/`
- 用户菜单项没有选中态，只有悬浮态，点击时弹出 dropdown，选项为 `退出登录`
- 菜单收起时，只保留图标

# 组件

- 侧边栏组件 Sider
- 菜单项组件 SiderMenu，通用菜单项抽离组件复用
